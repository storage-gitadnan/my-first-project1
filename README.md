# ☕ Afficionado Coffee Roasters – Data Analysis Project

## 📊 Project Overview

This project focuses on analyzing coffee sales data to uncover meaningful business insights.
It helps understand customer behavior, product performance, and sales trends.

---

## 🎯 Objectives

* Analyze sales data of coffee products
* Identify top-selling items
* Understand customer purchasing patterns
* Build an interactive dashboard for visualization

---

## 📁 Project Structure

* `dashboard.py` → Streamlit dashboard for visualization
* `methodology.py` → Data cleaning and processing logic
* `Afficionado Coffee Roasters.csv` → Dataset used for analysis
* `.gitignore` → Files ignored by Git

---

## 🛠️ Tools & Technologies

* Python 🐍
* Pandas
* Streamlit
* Data Visualization

---

## 📈 Features

* Sales trend analysis
* Product performance insights
* Interactive dashboard
* Clean and structured data processing

---

## ▶️ How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder

```
cd your-repo-name
```

3. Install required libraries

```
pip install -r requirements.txt
```

4. Run the Streamlit app

```
streamlit run dashboard.py
```
this is the link of live dashboard link
http://localhost:8501/
